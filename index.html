Getting started

Python is a high-level programming language that has many applications, including machine learning, data analysis, scripting, and back-end development. Now, at this point you might already be thinking that this sounds way to complicated already. However, this is where Python shines; Python is incredibly close to the human language.

For example, compare the images below. Both snippets of code perform the same task: printing "Hello World!". One image is written in Python, and one is written in Java.

This isn't to say that languages lower than Python don't have their uses; lower-level languages have greater performance, for example. However, this does show that Python is significantly easier to learn for beginners. Given that after learning one language it is a lot easier to learn another, its easy to see why Python is a great language to learn first. By learning Python first, you'll learn all the key concepts of programming, have a useful language at your disposal, and be able to learn all of this without spending ten hours memorising syntax!

Installing Python

Installing Python is pretty straight forward.

  1. Go to python.org and download the latest version of Python for your operating system.
  2. Run the executable/installer.
  3. Before clicking "Install Now", make sure you have checked the "Add Python … to PATH" checkbox.
  4. After clicking "Install Now", give administrator permissions when prompted and wait for Python to install.
If you cannot install Python for any reason, you can follow the guide along using repl.it, an online code editor; create a repl and choose Python as the language. You can ignore the rest of this installation guide.

Visual Studio Code: Best Python extensions

Although optional, the following extensions are available for free in Visual Studio Code (ctrl+shift+x to view extensions tab):

  • AREPL for python: AREPL allows you to view a window besides your code which will display the output of your program, as well as all variables and their values.
  • autoDocstring: this extension will automatically generate docstrings for Python functions, which will help with documentation.
  • Path Intellisense: this extension will suggest and auto-complete paths when programming, which will help make sure your correctly referencing other files.
  • Pylance/Python: usually installed after being prompted following Python detection, these extensions allow you to use Visual Studio Code to write in Python, properly.
  • python snippets: this extension features a collection of snippets of Python code, which will help speed up development.
  • Tabnine AI Autocomplete: this extension works with many languages. In short, it uses AI to predict what you wish to type; press TAB to confirm.
  • Python Indent: this extension will allow you to make optimal indents in Python, overriding Visual Studio Codes indents (which are pretty bad).
  • Better Comments: this extension will allow you to make better comments in Python, which will help with documentation.
  • Bracket Pair Colorizer 2: nested brackets can be quite confusing. Luckily, this extension gives them different colours, so you can identify them quickly.
You can use the F1 key to run extension commands; in general, type the extension name to view its commands.
